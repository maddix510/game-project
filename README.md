Quick start

- Install the Live Server extension in VS Code.
- Open this workspace and press "Go Live" (or open `index.html` in a browser).

Controls

- Click `Deploy Drone` to spawn an autonomous unit that gathers scrap.
- Click `Build Light` to place a light near the base (reduces night danger).
- Drones auto-gather from glowing resource nodes and return scrap to the base.
- Survive through nights; enemies spawn at night and chase the base and units.
